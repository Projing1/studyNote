# Vue-Press：

# 课程导学

什么是vue-presss：由两部分组成：第一部分是一个极简静态网站生成器，它包含由Vue驱动的主题系统和插件API，第二部分是为书写技术文档而优化的默认主题，它的诞生初衷是为了支持Vue及其子项目的文档需求。

# 设计——运行原理

1、什么是Vue-Press ：静态网站生成器，在MArkdown中使用Vue。0.X版本只有简单的主题系统，入口只能是一个SFC。1.X版本有插件系统，主题系统，和布局系统；
2、快速上手：yarn create vuepress。

# MarkDown

定义：是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。
用途：用于编写说明文档，电子书Gitbook，并以README.md的文件名保存在软件的目录下面。
写法：一个#代表H1标题，两个#代表H2标题……； + 代表无序列表； ``` ```包含着代码块； 表格单元格之间用 | 分割，表头下方用一行---分割，代码里的每行代表一个表格行 ； 图片插入语法：![描述](路径)；
对Markdown的扩展： vue-press;
